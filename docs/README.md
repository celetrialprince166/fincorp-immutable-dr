lab. Each chapter follows the same shape: **Goal → Prerequisites → Concepts (the why) →
Steps → Verification → Cost & teardown → Key takeaways.**

Screenshots live in [`assets/`](assets/), named `p<phase>-<subject>-NN.png`.

[`assets/fincorp-architecture.png`](assets/fincorp-architecture.png), generated from
[`architecture-diagram-prompt.md`](architecture-diagram-prompt.md).
